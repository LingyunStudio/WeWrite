# WeWrite 公众号编辑器示例

这是一篇覆盖所有 Markdown 格式的示例文档，用于演示 wewrite 主题在微信公众号中的完整渲染效果。

## 文本格式

### 行内格式

正文段落支持多种行内格式：**加粗文本**、*斜体文本*、***粗斜体***、~~删除线~~、`行内代码`、<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>R</kbd> 键盘符号，以及 [超链接](https://example.com)。

还可以混合使用：**加粗里有 *斜体* 和 `代码`**，*斜体里有 **加粗***，~~删除线里有 **加粗**~~。

### 上下标与公式

markdown 扩展语法：H~2~O 是水分子，2^10^ = 1024，E = mc^2^。

Latex 标准语法：$H_2O$ 是水分子，$2^{10} = 1024$，$E = mc^2$

行内公式：$E = mc^2$，$\alpha + \beta = \gamma$，$x^2 + y^2 = r^2$。

块公式：

$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$

$$
\int_0^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$$

### 长段落

这是一段较长的正文文字，用来测试正文在窄宽度（公众号 620px）下的换行效果。微信公众号正文宽度比 Typora 默认的 860px 要窄很多，所以主题里部分固定宽度已做自适应处理。这段话会自动换行成多行，行高 1.5 倍字号，阅读体验应该舒适。如果你在右侧预览区看到了正确的换行和行距，说明排版没有问题。

### Python

```python
# Python 代码示例
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

print(list(fibonacci(10)))
```
